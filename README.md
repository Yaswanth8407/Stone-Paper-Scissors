# ✊✋✌️ Stone Paper Scissors Game

A simple and interactive **Stone-Paper-Scissors web game** built using **HTML, CSS, and JavaScript**.
Play against a bot, track your score, and enjoy smooth animations and a modern UI.

---

## 🎮 Features

* Play **Stone, Paper, Scissors** against a bot
* **Random bot move generation**
* **Animated bot thinking effect**
* **Score tracking**

  * Wins
  * Losses
  * Ties
* **Popup result display**
* **Try Again button**
* **Reset Game button**
* **Modern UI with gradients and glassmorphism**
* **Hover animations and smooth transitions**

---

## 🛠 Technologies Used

* **HTML5** – Structure of the game
* **CSS3** – Styling, animations, and layout
* **JavaScript** – Game logic and DOM manipulation

## 🚀 How to Run the Project

1. Clone the repository

```
git clone https://github.com/Yaswanth8407/Stone-Paper-Scissors.git
```

2. Open the project folder.

3. Open **index.html** in your browser.

That's it! 🎉

---

## 🎯 How to Play

1. Click on one of the moves:

   * ✊ **Stone**
   * ✋ **Paper**
   * ✌️ **Scissors**

2. The bot will think for a moment and reveal its move.

3. The result will be shown in a popup:

   * 🏆 Win
   * ☠️ Lose
   * 🤝 Tie

4. Use:

   * **Try Again** → play another round
   * **Reset Game** → reset all scores

---

## 🧠 Game Logic

* Stone beats Scissors
* Scissors beats Paper
* Paper beats Stone
* Same move results in a **tie**
